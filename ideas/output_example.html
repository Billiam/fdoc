GET https://api.example.com/members/list

Request Parameters

Optional
  • limit

    Limits the number of results returned, used for paging.

    • Name: limit
    • Type: Integer
    • Required: false
    • Default: 50
    • Description: Limits the number of results returned, used for paging.
  • offset

    Offsets the results returned, used for paging.

    • Name: offset
    • Type: Integer
    • Required: false
    • Default: 0
    • Description: Offsets the results returned, used for paging.

Response Parameters

Optional
  • members
    • Name: members
    • Type: Array

Response Codes

Successful Response Codes
  • 200 OK

    A list of current members

PUT https://api.example.com/members/add

Request Parameters

Required
  • name

    The name of the member to add, must be unique.

    • Name: name
    • Type: String
    • Required: true
    • Description: The name of the member to add, must be unique.
    • Example: Max Power
Optional
  • email

    The email of the member to add

    • Name: email
    • Type: String
    • Required: false
    • Description: The email of the member to add
    • Example: mpower@example.com

Response Codes

Successful Response Codes
  • 200 OK

    Returns the member just added

Failure Response Codes
  • 400 Bad Request

    Member already exists
