» fdoc Sample API » Members API

The list of members, filtered by params

Request

Example Request 

{
  "limit": 50,
  "offset": 0,
  "older_than": "",
  "newer_than": ""
}

Request Parameters

  • limit

    Limits the number of results returned, used for paging.

    • Required: no
    • Type: integer
  • offset

    Offsets the results returned, used for paging.

    • Required: no
    • Type: integer
  • older_than

    If provided, results returned will only be updated before the given date (exclusive)

    • Required: no
    • Type: string
    • Format: date-time
  • newer_than

    If provided, results returned will only be updated after the given date (inclusive)

    • Required: no
    • Type: string
    • Format: date-time

Response

Example Response 

{
  "members": [
    {
      "name": "Captain Smellypants",
      "email": "smelly@pants.com",
      "member_since": "2012-01-01 13:00:00 UTC"
    }
  ]
}

Response Parameters

  • members
    • Required: no
    • Type: array
    • Items

      Representation of a member

      • Required: no
      • title: member
      • name

        Member’s name

        • Required: yes
        • Type: string
        • Example: "Captain Smellypants"
      • email

        Member’s email

        • Required: no
        • Type: string
        • Example: "smelly@pants.com"
      • member_since

        The date the member joined

        • Required: no
        • Type: string
        • Format: date-time

Response Codes

Successful Response Codes

  • 200 OK

Failure Response Codes

  • 400 Bad Request
