» fdoc Sample API » Members API

The list of members, filtered by params

Request

Example Request 

Accept: application/json
{
  "limit": 50,
  "offset": 0,
  "older_than": "",
  "newer_than": ""
}

Request Headers

  • Accept

    Acceptable media types

    • Required: no
    • Type: string
    • Example: "application/json"

Request Parameters

  • limit

    Limits the number of results returned, used for paging.

    • Required: no
    • Type: integer
  • offset

    Offsets the results returned, used for paging.

    • Required: no
    • Type: integer
  • older_than

    If provided, results returned will only be updated before the given date (exclusive)

    • Required: no
    • Type: string
    • Format: date-time
  • newer_than

    If provided, results returned will only be updated after the given date (inclusive)

    • Required: no
    • Type: string
    • Format: date-time

Response

Example Response 

Content-Type: application/json
{
  "members": [
    {
      "name": "Captain Smellypants",
      "email": "smelly@pants.com",
      "member_since": "2012-01-01 13:00:00 UTC"
    }
  ]
}

Response Headers

  • Content-Type

    Response format

    • Required: yes
    • Type: string
    • Example: "application/json"

  • members
    • Required: no
    • Type: array

        • Required: no
        • Type: string

      • member_since
