Add a new member

Request

Example Request 

Accept: application/json, text/html
{
  "name": "Max Power",
  "email": "mpower@example.com"
}

Request Headers

  • Accept

    Acceptable media types

    • Required: no
    • Type: string
    • Example: "application/json, text/html"

Request Parameters

  • name

    The name of the member to add, must be unique.

    • Required: yes
    • Type: string
    • Example: "Max Power"
  • email

    The email of the member to add

    • Required: no
    • Type: string
    • Example: "mpower@example.com"

Response Codes

Successful Response Codes

  • 200 OK

    Returns the member just added

Failure Response Codes

  • 400 Bad Request

    Member already exists
